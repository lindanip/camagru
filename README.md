# Camagru
This web project is about creating a small web application that allows a user to make basic photo and video editing using their webcam and some predefined images.

## Prerequisites
1. Web Server.
2. Relational Database management system.
3. PHP

## Set up
1. clone the this project to where your web server finds web pages to serve. 
1. Run the setup.php in the browser to create the database and the tables requrired.
    example: [http://localhost/camagru/camagru/config/setup.php](http://localhost/camagru/camagru/config/setup.php). 
 
2. Then the user can go to the index page : [http://localhost/camagru/camagru/index.php](http://localhost/camagru/camagru/index.php) to either login or sign up for a new account.
